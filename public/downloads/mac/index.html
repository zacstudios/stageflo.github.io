StageFlo for macOS

This page is the stable macOS download entry for StageFlo. The private desktop build pipeline publishes the latest mac installer files here.

Download macOS Universal DMG (Intel + Apple Silicon)

After the first desktop publish run, mac files will be available under paths such as /downloads/mac/ and the update feed will be available at /updates/.

If you do not see the DMG yet, the publish workflow has not completed successfully yet.
